abstract class Futures::Try(T)

Overview

Try represents a computation that may either result in an exception, or return a value. Instances of Try(T) are either an instance of Success(T) or Failure(T) Eg.

a = Success(Symbol).new :hello
a.get   # => :hello
a = Failure(Symbol).new(Exception.new "Error")
a.get   # => Raises Exception("Error")
# Pattern match over a Try(T)
case a
when Success
  # do something with result
when Failure
  # handle exception
end
